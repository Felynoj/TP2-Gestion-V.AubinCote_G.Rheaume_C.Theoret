Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Il n'y avait aucun cours théorique durant le module de Projet Web. Par contre, certains d'entres nous ont du pousser nos recherches pour pouvoir avancer dans nos projets, car certaines facettes nécéssitaient l'ajout de contenu pas encore vu dans le programme.

Pratiques (quels sont les logiciels que vous avez appris)?

Les mêmes logiciels ont été utilisés qu'à l'habitude dont Adobe XD pour la maquette, et le début de l'intégration d'APIs pour quelques un avec celui de Google Maps.

Conditionnelles (quels sont les projets que vous avez réalisés)?

Un site Web de restauration quasi complet avec un menu, une infolettre, une section contact, et une interface d'administration.
