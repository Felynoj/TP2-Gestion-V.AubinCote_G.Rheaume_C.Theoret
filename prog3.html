Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Le cours Cadriciel Web fait partie du champ de compétence Programmation et initie l'étudiant à un cadriciel Web. L'étudiant acquiert des connaissances techniques reliées l'utilisation d'un cadriciel dans un contexte de manipulation de bases de données et de production de site Web dynamique, principalement du côté serveur. Il acquiert aussi des notions liées à la validation et des interfaces en ligne de commande.

Pratiques (quels sont les logiciels que vous avez appris)?

  • Compréhension adéquate du devis de production
  • Installation et mise en place adéquate du cadriciel
  • Utilisation adéquate des fonctionnalités du cadriciel pour traiter les requêtes HTTP
  • Utilisation adéquate des fonctionnalités du cadriciel pour effectuer le traitement et l'affichage des données
  • Fonctionnement correct du produit réalisé
  • Qualité du code et documentation : Le code est bien présenté et documenté; la programmation et la nomenclature respectent les standards.

Conditionnelles (quels sont les projets que vous avez réalisés)?

Nous avons réaliser quelques projets sympathiques pour mettre en application nos nouvelles connaissances, dont un à travers le module au complet qui se trouve être une interface de nouvelles.
